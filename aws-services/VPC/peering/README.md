# VPC — Virtual Private Cloud

Serviço da AWS para criação e gerenciamento de redes virtuais isoladas na nuvem.

## Conceitos Principais
- Rede virtual privada
- Isolamento lógico
- Controle total de IPs e sub-redes
- Integração com serviços AWS

## Componentes
- CIDR Block
- Subnets (Public / Private)
- Route Tables
- Internet Gateway
- NAT Gateway
- Security Groups
- Network ACLs

## Funcionalidades
- Comunicação entre recursos AWS
- Controle de tráfego de entrada e saída
- Segmentação de rede
- Alta disponibilidade por AZ

## Casos de Uso
- Hospedagem de aplicações
- Ambientes corporativos
- Arquiteturas multi-tier
- Infraestrutura segura na nuvem

## Boas Práticas
- Separar subnets públicas e privadas
- Usar múltiplas AZs
- Restringir tráfego com Security Groups
- Planejar corretamente o CIDR
